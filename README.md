# Music-Genre-Classification-using-Deep-Learning
A deep learning-based music genre classification system using TensorFlow and Librosa. It extracts MFCC features from audio files and trains a CNN model to classify genres. Includes a Streamlit web app for real-time prediction by uploading .wav files with confidence scores.
